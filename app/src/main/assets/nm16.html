17. ГАММА-ФУНКЦИЯ, ОПРЕДЕЛЕНИЕ, СВОЙСТВА, ВЫЧИСЛЕНИЕ

Гамма-функция - одна из самых важных неэлементарных функций анализа. Это математическая функция, которая расширяет понятие факториала на поле комплексных чисел. Обычно обозначается Г(z). Является мероморфной на комплексной плоскости и имеет полюса в точках:

z=0,-1,-2,-3,...

(Функция комплексной переменной называется мероморфной, если она определена на всей комплексной плоскости и не имеет в конечной части плоскости особых точек, отличных от полюсов.)

Гамма функция находит очень широкое применение в прикладном анализе. В статистике широко используется гамма-распределение, частными случаями которого являются экспоненциальное распределение и распределение хи-квадрат. В данной главе вводится понятие гамма-функции, приведены ее основные свойства, а также показан алгоритм ее численного расчета.

На всю комплексную плоскость гамма-функция аналитически продолжается через тождество:

Г(z+1)=z⋅Г(z)

Это равенство можно доказать, применяя интегрирование по частям к интегральному определению гамма-функции, тождество верно для любого комплексного значения z.

Данное рекуррентное соотношение является очень важным и используется при расчете гамма-функции.

Теперь рассмотрим очень важный вопрос, касающийся расчета гамма-функции. Для этого рассмотрим несколько возможных интервалов аргумента z.

Пусть 0 < z < 1, тогда в соответствии со свойствами гамма-функции можно вычислить:

Г(z) = Г(z+1) / z

где 1 < z+1 < 2, другими словами значение гамма функции при 0 < z < 1 может быть вычилено через значения гамма функции при 1 < z < 2.

Пусть z > 2, тогда для вычисления значения гамма-функции можно записать:

Г(z) = (z-1)⋅Г(z-1)

При этом Г(z-1) = (z-2)⋅Г(z-2), и если продолжать, то можно свести к вычислению гамма-функции в интервале 1 < z < 2.

Рассмотрим на примере:

Г(4,6)=Г(1+3,6) = 3,6⋅Г(3,6)=3,6⋅Г(1+2,6)=...

... = 3,6⋅2,6⋅Г(2,6) = 3,6⋅2,6⋅Г(1+1,6) = 3,6⋅2,6⋅1,6⋅Г(1,6)

То есть опять задача свелась к вычислению гамма-функции в интервале 1 < z < 2.
